# Contributing to Dimes Multiply SDK

Thanks for your interest in contributing to the Dimes Multiply TypeScript SDK!

## Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/<your-username>/multiply-sdk.git`
3. Install dependencies: `npm install`
4. Create a feature branch: `git checkout -b feat/your-feature`

## Development

- Run tests: `npm test`
- Build: `npm run build`
- Lint: `npm run lint`

## Pull Requests

- Keep PRs focused on a single change
- Include tests for new functionality
- Update documentation if applicable
- Follow the existing code style

## Reporting Issues

Open an issue on GitHub with:
- A clear description of the problem
- Steps to reproduce
- Expected vs actual behavior
- SDK version and environment details
